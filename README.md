### 谁适合看？

记录这个东西希望可以帮助到大家入门，为 IT 行业贡献出自己的一份绵薄之力，但是深知个人精力有限，难免有遗漏或者错误的地方。

有些内容过于啰嗦，并不是不想概括着写，而是本人在学习的过程中发现，对于一个新手来说，太简要反而会使得新手有些懵。因此，很多知识点我尽量介绍详细一点。

同时欢迎各位在此多多提意见，我是个很喜欢别人提意见的人，那样才能够意识到自己哪里不足，才能慢慢改进。
